# Projet L2-Chloé

This projet is for fun only. Based on L2J & L2JUnity.

## Getting started

This is what you need :

* Lineage 2 High Five Client
* JDK 11 or newer
* MySQL 5.7 or newer

## Technology used

* Maven
* Hibernate
* Log4j2
* Guice

## How it works

This projet is the "login part" of Lineage 2 emulator servers.
